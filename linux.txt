pwd-------->The pwd command is used to display the location of the current working directory.
syntax: pwd
mkdir------->The mkdir command is used to create a new directory under any directory.
syntax: mkdir directoryname
rmdir-------->The rmdir command is used to delete a directory.
syntax:rdmir directoryname
ls-------->this is used to displaythe list of content of a directory
syntax: ls
cd-------->The cd command is used to change the current directory.
syntax: cd directory name
touch----->The touch command is used to create empty files. We can create multiple empty files by executing it once.
syntax: touch filename or touch file1 file2 etc
cat------>the cat command  is a multi-purpose utility in the Linux system. It can be used to create a file, display content of the file, copy the content of one file to another file, and more.
syntax: cat> filename;
to display the content: cat filename
rm-------->The rm command is used to remove a file.
syntax: rm filename
cp-------->The cp command is used to copy a file or directory.
syntax: cp existing filename  new filename
mv -------->The mv command is used to move a file or a directory form one location to another location.
syntax: mv filename directory path
rename------>The rename command is used to rename files. It is useful for renaming a large group of files.
syntax: rename 's/old-name/new-name/'file
